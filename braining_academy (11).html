📋 Firebase 무료 설정 방법 (5분 소요)
1️⃣ console.firebase.google.com 접속 → Google 계정으로 로그인
2️⃣ 프로젝트 만들기 → 이름 입력 (예: 브레이닝학원) → 만들기
3️⃣ 왼쪽 메뉴 빌드 → Realtime Database → 데이터베이스 만들기 → 테스트 모드 선택
4️⃣ ⚙️ 프로젝트 설정 → 내 앱 → 웹(</>) 클릭 → 앱 등록
5️⃣ 표시된 firebaseConfig 전체 복사 → 아래 붙여넣기
※ 같이 쓸 모든 선생님에게 이 설정값을 공유하세요. 같은 값을 입력해야 함께 쓸 수 있습니다.
Firebase 콘솔 → 빌드 → Realtime Database → 상단에 표시된 URL을 복사해서 입력하세요.
데이터는 Firebase 서버에 저장됩니다. 같은 설정값을 가진 기기만 접근 가능합니다.
인터넷이 없을 때도 기기에 저장된 데이터로 계속 사용할 수 있습니다.
